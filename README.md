# study-apps

生成AI(Claude)との対話で自作した、自分専用の学習アプリ集。

## 公開ページ(ここから開く)

**https://huangwox-prog.github.io/study-apps/**

## 収録アプリ

- [数Ⅰ特訓帳](https://huangwox-prog.github.io/study-apps/sugaku1-tokkun.html) — 数と式・集合と命題 全31テーマ93問、苦手復習機能つき
- [有理化道場](https://huangwox-prog.github.io/study-apps/yurika-dojo.html) — ルートの有理化をゼロから段階的に
- [一次不等式特訓](https://huangwox-prog.github.io/study-apps/ichiji-futoshiki.html)
- [絶対値特訓](https://huangwox-prog.github.io/study-apps/zettaichi.html)

基本情報技術者(FE)科目B の模擬試験アプリも順次追加予定。

## 目的

総合型選抜の数学Ⅰ口頭試問対策と基本情報技術者試験の学習のため、
自分の苦手に合わせた反復練習アプリをAIとの対話で作り、日々の学習に使っている。
